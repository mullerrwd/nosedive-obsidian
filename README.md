# nosedive-obsidian
A CSS theme for obsidian.md based on the Fictional UI shown in the episode ["Nosedive" from Black Mirror](https://www.gemmagrace.co.uk/NOSEDIVE-2016). It can be toggled between a light theme and a dark theme.
